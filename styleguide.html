Every section of the page starts with a div class that defines what kind of section it is. This one is called .text-samples.

Inside of text-samples is a div with either .white-bg or .blue-bg.

Inside that div is .container, which adds a max-width and margin: 0 auto.

You can also add .section-border class to the .container div (class="container section-border") to add a black border like the one you see around this entire section. You can do that for any type of sections and the internal contents will adjust.

That's the basic setup for a given section. Follow this each and every time you make something.

This section is called image-pullquotes.

Inside of .container you need to create more div called .flex. Then you can get started - it creates a flexbox container that can manage multiple columns. Flex starts right here:

Here's how you make the images and captions work.

Each image + caption pair (contained within a figure element) has three classes:

  1. .image-pullquote: This gives the image/caption its look.
  2. .left or .right: decides whether the caption is offset to the left or to the right. It defaults to left, but you should specify it.
  3. .flex-1-of-2: This makes flexbox work. Use this for 2 column layouts, .flex-1-of-3 for 3 column layouts, etc, all the way up to 5. If you need columns that aren't all the same size you'll need to override these.

Here's how you make these text boxes work.

Very simple: just give them the same .flex-1-of-2 class. That's it.

You can use flexbox's order property (Google it!) to rearrange the order of items on mobile.
